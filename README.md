# Unique Green 營運管理系統

加盟連鎖飲料品牌營運儀表板，支援 Excel 資料匯入、PSD 基準分析、異常警示。

## 專案結構

```
ug-dashboard/
├── index.html          ← 首頁儀表板
├── css/
│   └── main.css        ← 共用樣式
├── js/
│   ├── shared.js       ← 共用邏輯（資料、警示計算）
│   └── excel.js        ← Excel 匯入引擎
├── pages/
│   ├── alerts.html     ← 異常警示
│   ├── stores.html     ← 門市列表（督導篩選）
│   ├── psd.html        ← PSD 基準明細
│   ├── settings.html   ← 管理後台
│   └── import.html     ← Excel 資料匯入
└── data/               ← 預留：未來放靜態 JSON
```

## 部署到 GitHub Pages

1. 建立新的 repository（Public）
2. 把整個資料夾的檔案上傳
3. Settings → Pages → Branch: main → Save
4. 等約 1 分鐘，網址為 `https://帳號.github.io/ug-dashboard/`

## Excel 匯入格式

| 欄位 | 必填 | 說明 |
|------|------|------|
| 門市名稱 | ✓ | 門市完整名稱 |
| 區域 | ✓ | 北部/中部/南部/東部 |
| 今日營收 | ✓ | 當日實際營收（整數） |
| 週日PSD | 建議 | 前月所有週日均值 |
| 週一PSD | 建議 | 前月所有週一均值 |
| 週二PSD | 建議 | 前月所有週二均值 |
| 週三PSD | 建議 | 前月所有週三均值 |
| 週四PSD | 建議 | 前月所有週四均值 |
| 週五PSD | 建議 | 前月所有週五均值 |
| 週六PSD | 建議 | 前月所有週六均值 |

> 在「匯入資料」頁面點「下載範本」可取得標準格式的 Excel 範本。

## 資料儲存說明

- 所有資料存在瀏覽器的 `localStorage`，關閉後不會消失
- 每次上傳新的 Excel 會覆蓋舊資料
- 如需重置，到「管理後台」點「重置資料」

## 功能說明

| 頁面 | 功能 |
|------|------|
| 儀表板 | 今日總覽、營收走勢、門市排行 |
| 異常警示 | 依閾值顯示紅/黃警示門市 |
| 門市列表 | 督導複選門市、個別比對 |
| PSD 明細 | 各店七天 PSD 基準查詢 |
| 管理後台 | 調整閾值、通知設定 |
| 匯入資料 | 拖曳上傳 Excel、下載範本 |
